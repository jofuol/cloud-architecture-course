# **Google Cloud CLI**

*Google Cloud Command Line Interface (gcloud CLI)* is the command-line tool for Google Cloud and It comes pre-installed on Cloud Shell and supports tab-completion

Use `gcloud` to manage your Google Cloud resources and services directly from the terminal.

!!! Note

    For full documentation of `gcloud`, in Google Cloud, refer to:

    - [the gcloud CLI documentation][gcloud-cli-documentation]{:target="_blank"}.
    - [the gcloud CLI overview guide][gcloud-cli-overview-guide]{:target="_blank"}.

This guide resume some of the most common `gcloud` commands to:

- [Managing accounts][accounts].
- [Managing projects][projects].
- [Managing regional and global configurations][regional-global-config].
- [Managing Virtual Machines][virtual-machines].
- [Managing Firewall Rules][firewall-rules].
- [Managing Load Balancing Service][load-balancing-service].
